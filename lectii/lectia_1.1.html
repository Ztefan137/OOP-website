Ce este POO?

Programarea Orientată pe Obiecte (POO) este o paradigmă de programare care utilizează obiecte și clase pentru a structura codul într-un mod care mimează lumea reală. În loc să ne concentrăm doar pe funcții repetitive, în POO organizăm logica în jurul datelor reale.

Clasa

Este tiparul sau schița (blueprint). Ea definește proprietățile și comportamentele comune pe care le vor avea obiectele. Nu ocupă memorie direct pentru date.

Obiectul

Este instanța concretă a unei clase. Reprezintă entitatea fizică creată, care are valori specifice pentru atribute și ocupă spațiu în memoria RAM.

Exemplu practic în C++

main.cpp
C++ 
#include <iostream>
using namespace std;

class Student {
public:
    string nume;
    int varsta;
};

int main() {
    Student s1;
    s1.nume = "Andrei";
    s1.varsta = 20;
    return 0;
}
