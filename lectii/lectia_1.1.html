Introducere în Programarea Orientată pe Obiecte

Înțelegerea trecerii de la paradigma procedurală la cea orientată pe obiecte.

Înapoi la Dashboard

▪ Ce este POO?

Programarea Orientată pe Obiecte (POO) este o paradigmă de programare care utilizează obiecte și clase pentru a structura codul într-un mod care mimează lumea reală. Spre deosebire de programarea procedurală, unde accentul se pune pe funcții și secvențe de pași, în POO ne concentrăm pe datele pe care vrem să le manipulăm și pe interacțiunile dintre acestea.

▪ Conceptul de Clasă vs. Obiect

Clasa

Este tiparul sau schița (blueprint). Ea definește proprietățile (atributele) și comportamentele (metodele) comune pe care le vor avea toate obiectele create pe baza ei. Nu ocupă memorie direct pentru date.

Obiectul

Este instanța concretă a unei clase. Reprezintă entitatea fizică sau conceptuală creată din acea clasă, care are valori specifice pentru atribute și ocupă spațiu în memoria RAM.

▪ Exemplu practic în C++

Priviți structura de bază a definirii unei clase și instanțierea unui obiect:

main.cpp C++ 
#include <iostream>
using namespace std;

class Student {
public:
    string nume;
    int varsta;

    void saluta() {
        cout << "Salut! Numele meu este " << nume << endl;
    }
};

int main() {
    Student s1;          // Instanțierea unui obiect
    s1.nume = "Andrei";
    s1.varsta = 20;
    
    s1.saluta();        // Apelarea metodei
    return 0;
}
